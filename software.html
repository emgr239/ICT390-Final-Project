Operating Systems

System software that manages computer hardware and software resources and provides common services for computer programs.

Linux Operating System

Linux is an open source operating system built around a common kernel. There are many different types of Linux distributions. If you are looking to build a desktop environment, GNOME and KDE are the two best Linux distributions you can use.

Linux allows a user to create and edit the environment more than most any other operating systems. Linux's code may be modified to the user's preferance.

Linux is a good choice for programming intensive and computer savy people, if you are looking to become a beteter computer expert give Linux a try!

Microsoft Windows Operating System

Microsoft Windows, or Windows, is a desktop environment. Windows was one of the first GUIs or Graphical User Interfaces, to hit the market.
