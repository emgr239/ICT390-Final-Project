Operating Systems

System softwares that manages computer hardware and software resources and provides common services for computer programs.

Linux Operating System

Linux is an open source operating system built around a common kernel. There are many different types of Linux distributions. If you are looking to build a desktop environment, GNOME and KDE are the two best Linux distributions you can use.

Linux allows a user to create and edit the environment more than most any other operating systems. Linux's code may be modified to the user's preferance.

Linux is a good choice for programming intensive and computer savy people, if you are looking to become a beteter computer expert give Linux a try!

Microsoft Windows Operating System

Microsoft Windows, or Windows, is a desktop environment. Windows was one of the first GUIs or Graphical User Interfaces, to hit the market.

Windows allows the user to customize some items of the interface but unless one get's Windows Professional, which comes with more features for those who know what they are doing.

Windows is widely used and a very common operating system. If you are planning on going into the industry that uses a pre-installed operating system, Windows is the way to go!

Audio Players

Softwares that plays and organizes music or other audio

Rhythmbox is an audio player for Linux. Most commonly used with the GNOME interface. Rhythmbox features include;

  • Music Playback
  • Gapless Playback
  • Music Import
  • Audio CD Burning
  • Album Cover Display
  • Song Lyric Display
  • Web Remote Control
  • And More!

Windows Media Playeris an audio and video player for Windows. Auto-installed with most windows OS. WMP features include;

  • Playback, audio, videos, etc.
  • Visualizations
  • Format Support
  • Burning and Ripping
  • Shell Integration
  • Online Features
  • Media Streaming
  • And More!
